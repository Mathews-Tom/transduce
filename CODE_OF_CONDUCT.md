# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1, as its code of conduct.

## Scope

Applies to all project spaces — repository issues and pull requests, code review, mailing lists, chat channels, and any public spaces where an individual is representing the project or its community.

## Reporting

Report concerns to the maintainers via the contact channel listed in [`SECURITY.md`](SECURITY.md). Reports are reviewed and investigated promptly and confidentially.

## Enforcement

Maintainers follow the Contributor Covenant v2.1 enforcement guidelines. Sanctions range from a private warning through temporary or permanent removal from project spaces, applied proportionally to the impact and recurrence of the conduct.
